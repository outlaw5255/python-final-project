# Small Business Inventory Tracker

#### PowerPoint prestation 

#### Description:

This project is a Python-based inventory management system created for a small business. The purpose of the program is to help users track products, quantities, and basic stock movement in an organized way. Many small businesses need a simple and affordable method to manage inventory, and this project demonstrates how Python can be used to solve that problem.

The main file in this project is TermProject.py. This file contains all of the Python code used to run the application. It includes a main() function along with several additional functions such as adding items, viewing inventory, updating stock, and recording sales. These functions are organized separately so the code is easier to read, maintain, and improve in the future.

The requirements.txt file is also included in this project. This file explains that no external libraries are required. The project uses standard Python features only, which makes it easy to run on most systems without installing extra packages.

This project uses a menu-driven design. When the user starts the program, they are shown a list of numbered options. The user can choose to view inventory, add new items, update stock quantities, record a sale, or exit the program. This type of interface was selected because it is simple, beginner-friendly, and easy to understand.

Inventory items are stored using dictionaries. Each item can contain a quantity, price, and reorder level. Dictionaries were chosen because they allow data to be stored clearly using labels. This helps keep product information organized and easy to access inside the program.

One important design decision was separating the program into multiple functions instead of writing everything in one block of code. This makes debugging easier and follows good programming practices. Another important decision was creating a text-based application instead of a graphical application. A text-based system is faster to develop, easier to test, and still demonstrates strong Python fundamentals.

This project is more advanced than a lab assignment because it combines many concepts into one complete application. These concepts include functions, loops, dictionaries, conditionals, user input, and program organization. It also simulates a real-world business need instead of solving only one small task.

If this project were expanded in the future, features such as file saving, employee login accounts, sales reports, or barcode scanning could be added. This shows how the project can grow into a more advanced business tool.

 This project demonstrates practical Python programming skills while creating something useful and realistic for a small business environment.
